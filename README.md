# 📰 Pulsar News

Portal de notícias moderno, responsivo e otimizado, desenvolvido para a publicação de conteúdos informativos, atualidades, desporto, tecnologia, política, economia e entretenimento.

---

## 📝 Sobre o Projeto

O **Pulsar News** é uma plataforma de notícias criada com o objetivo de oferecer informação rápida, organizada e totalmente acessível em qualquer tipo de dispositivo. O projeto foca-se na melhor experiência de leitura para o utilizador e na máxima eficiência técnica para SEO e monetização.

## 🚀 Funcionalidades

- **Design 100% Responsivo:** Otimizado para visualização impecável em smartphones, tablets e computadores.
- **Destaques em Evidência:** Secção dedicada para as notícias principais do dia.
- **Categorização Organizada:** Separação fluida de conteúdos por temas (Desporto, Política, Entretenimento, etc.).
- **Otimização para SEO:** Estrutura de tags e metadados planeada para uma excelente indexação nos motores de busca (Google).
- **Pronto para Google AdSense:** Espaços publicitários estrategicamente posicionados e em conformidade com as diretrizes de monetização.
- **Carregamento Rápido:** Código limpo e leve para garantir uma navegação veloz.
- **Fácil Personalização:** Estrutura modular simples de editar e adaptar para outros nichos.

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as tecnologias web fundamentais:

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização moderna, layouts fluídos e responsividade.
- **JavaScript:** Interatividades e dinamismo na página.
